Customer Churn Analysis
Objective

Analyze customer churn behavior to identify key factors influencing customer retention and highlight high-risk segments using data visualization and exploratory analysis.

Overview

This project explores demographic, contractual, and financial customer data to understand churn patterns. Clear and informative charts are used to communicate insights effectively.

Key Insights

Senior citizens show higher churn rates

Customers with shorter tenure are more likely to churn

Month-to-month contracts have the highest churn

Long-term contracts improve customer retention

Tools Used

Python

Pandas, NumPy

Matplotlib

Outcome

The analysis provides actionable insights that can help businesses reduce churn through targeted engagement and improved contract strategies.
